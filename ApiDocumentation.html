API Documentation

  • POST


    • /databaseFunctionsApi/newId/[id]
    [id] - o valoare numerica generata in mod unic de account service creeaza un nou spatiu de stocare pentru un user Raspuns: JSON: { "response": "succes" } sau { "response": [error] }
    /databaseFunctionsApi/databases/[id]/[database_name]
    creeaza o noua baza de date Raspuns: { "response": "created" } sau { "response": [error] }
    /databaseFunctionsApi/createTable/[id]/[database_name]
    Example body: { "tableName":"pls4", "numberColumns":2, "columnDescription": [ { "name":"bubu", "type":"varchar", "length":"10", "null":"0" }, { "name":"bubu2", "type":"varchar", "length":"10", "null":"1" } ] } Raspuns: { "response": "Succes" } sau { "response": [error] }
    /databaseFunctionsApi/createFunction/[id]/[database_name]/
    Example body: { "functionName":"hehee", "numberOfParameters":1, "returnType":"INT", "parameterDescription": [ { "name":"bubu", "type":"CHAR(34)" } ], "bodyText":"BEGIN\r\nRETURN 1;\r\nEND" } Raspuns: { "response": "Succes" } sau { "response": [error] }
    app/generateCodeService/api/apiGC.php
    { "id":"12", "language":"Java", "data":"SELECT * FROM hihi" }
    { "response": "public class jdbcConn {\r\n public static void main(String[] args) throws Exception{\r\n Class.forName(\"org.apache.derby.jdbc.ClientDriver\");\r\n Connection con = DriverManager.getConnection\r\n (\"jdbc:derby://localhost:80/12\",\"root\",\"\");\r\n Statement stmt = con.createStatement\r\n (ResultSet.TYPE_SCROLL_SENSITIVE,\r\n ResultSet.CONCUR_UPDATABLE);\r\n con.setAutoCommit(false);\r\n ResultSet rs = null;\r\n \r\n rs = stmt.executeQuery(\"SELECT * FROM hihi\");\r\n rs.last();\r\n System.out.println(\"output is \"+ rs.getRow());\r\n }\r\n }" }

  • GET


    • ⦁ /databaseFunctionsApi/databases/[id]
    Returneaza numele bazelor de date create de utilizatorul cu id-ul [id] Raspuns: [ { "database_name": "" } ... ]
    ⦁ /databaseFunctionsApi/functions/[id]/[database_name]
    Exemplu de raspuns: [ { "function_name": "dae", "function_body": "BEGIN\nRETURN 1;\nEND" }, { "function_name": "ds1", "function_body": "BEGIN\nRETURN 1;\nEND" }, ]
    ⦁ /databaseFunctionsApi/tables/[id]/[database_name]
    Exemplu de raspuns: [ { "table_name": "gouda" }, { "table_name": "tortillas" }, ]
    ⦁ databaseFunctionsApi/procedures/[id]/[database_name]
    Exemplu de raspuns: [ { "procedure_name": "country", "procedure_definition": "BEGIN\r\n SELECT Name, HeadOfState FROM Country\r\n WHERE Continent = con;\r\nEND" }, { "procedure_name": "country2", "procedure_definition": "BEGIN\r\n SELECT Name, HeadOfState FROM Country\r\n WHERE Continent = con;\r\nEND" } ]
    ⦁ databaseFunctionsApi/structure/[id]/[database_name]/[table_name]
    Exemplu de raspuns: [ { "name": "hghv", "type": "varchar", "length": "5", "null": "NO" } ]

  • DELETE

    • ⦁ /databaseFunctionsApi/databases/[id]/[database_name]
    Raspuns: { "response": "succes" } sau { "response": [error] }
    ⦁ /databaseFunctionsApi/tables/[id]/[database_name]
    Exemplu body: { "table_name":"pls3" } Raspuns: { "response": "succes" } sau { "response": [error] }
    ⦁ /databaseFunctionsApi/column/[id]/[database_name]/[table_name]
    Exemplu de body: { "name":"column_name" } Raspuns: { "response": "succes" } sau { "response": [error] }

  • PUT

    • /databaseFunctionsApi/mainpage/save/[id]/[database_name]
    Salveaza text in spatiul de stocare al userului corespunzator bazei de date mentionate Exemplu de body: { "schema":"DROP TABLE tab", "query":"SELECT * FROM tabl" } Raspuns: { "response": "Saved" } sau { "response": [Error] }
    /databaseFunctionsApi/mainpage/[id]/run
    Se ruleaza codul sql dat in body si se returneaza raspunsul la interogare sau o eroare corespunzatoare Exemplu de body: INSERT INTO haha(ha) VALUES(1); Exemplu de raspuns: { "response": [Response] } { "response": ["Error:] }
    databaseFunctionsApi/selectData/[id]/[database_name]/[table]
    Realizazea un select simplu pe o tabela data si returneaza raspunsul sau o eroare Exemplu de body: { "columns":"col1,col2", "where":"", "group":"", "having":"" } Exemplu de raspuns: { "response": "Unknown column 'col1' in 'field list'" }
    databaseFunctionsApi/insertData/[id]/[database_name]/[table]
    Insereaza valori in tabel corespunzatoare datelor din body Exemplu de body: { "columns":"col1, col2", "values":"1,2" } Raspuns: { "response": "Succes" } sau { "response": [Error] }
